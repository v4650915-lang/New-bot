# Telegram Бот с Платной Подпиской

## Возможности
- Регистрация пользователей
- Платная подписка (30/90/365 дней)
- Инлайн кнопки для навигации
- Доступ к приложению после оплаты
- Проверка статуса подписки
- Автоматическое истечение подписки

## Установка

1. Скопируйте `.env.example` в `.env` и заполните свои данные:
```bash
cp .env.example .env
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Запустите бота:
```bash
python main.py
```

## Команды

- `/start` - Начало работы
- `/profile` - Мой профиль и подписка
- `/subscribe` - Купить подписку
- `/app` - Получить доступ к приложению
- `/help` - Помощь

## Настройка платежей

1. Получите токен платежного провайдера в @BotFather
2. Укажите его в `.env` как `PAYMENT_PROVIDER_TOKEN`
